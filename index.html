Introduction
Hunyuan-Game is an AIGC-based toolset for boosting development process in gaming industry. It's consisted by four image generation models and five video generation models. These models address significant challenges in game development and are highly adaptable to various game scenarios.
Hunyuan-Game-Image Generation
Built upon a customized image generation model tailored for gaming scenarios, Hunyuan-Game-Image can provide various AIGC-based tools for 2D art design in gaming industry.
Text-to-Image Generation
The model is designed specifically for game scenarios, integrating deep knowledge of the game to enhance aesthetic expression and systematically understand diverse art styles. We introduce a prompt optimization model to improve image quality and provide precise semantic descriptions, lowering the barrier of professional use.

Game Character Generation
Game Character Generation model enhances control and professionalism through its structured workflow. Its specialized character consistency model further ensures coherent character representation across poses by analyzing structural maps (white models/depth maps), overcoming the critical limitation of inconsistent style and settings in conventional T2I systems. the solution also establishes a standardized pipeline for industrial-scale game production, effectively bridging the gap between creative concepts and production-ready assets.
Hunyuan-Game-Video Generation
Hunyuan-Game-Video Generation is a video generation toolset designed specically for gaming industry. Built upon Hunyuan I2V base model, this toolset provides many useful functions for 2D and 3D arts in game production pipeline.
Image-to-Video Generation
This model has demonstrated exceptional performance in the field of game video generation, surpassing the previous methods, such as Kling and Wan, in visual fidelity and temporal consistency. Moreover, the I2V model can serve as a foundation for many downstream tasks, thereby providing a robust fundamental capability.

360° A/T Pose Character Video Generation
As the first model of its kind in the industry, it allows for the creation of 360-degree rotation videos from any character illustration, ensuring A/T-pose standards. The generated video helps to eliminate visual blind spots from a single perspective, allowing the designer to evaluate the character design from different views.
Dynamic Illustration Generation
This model introduces the ability to generate dynamic illustrations, creating seamless looping animations from static character illustrations while maintaining high temporal consistency. It meets the demand for smooth and natural character movements in game animations. Dynamic Illustration has a wide range of applications, such as character entry animations and promotional videos.
Generative Video Super-Resolution
Our video super-resolution model is exceptional in the game and anime fields, transforming low-resolution assets into impressive 2K videos. This greatly improves visual quality without sacrificing detail, which is crucial for preserving the aesthetic quality of game visuals.
Interactive Game Video Generation
The proposed model addresses key limitations in existing world modeling approaches by offering several innovative advantages. First, it unifies diverse keyboard/mouse inputs (e.g., W, A, S, D, Space) into a shared camera representation space, enabling fine-grained, temporally coherent control over game scene video synthesis while ensuring physical plausibility during cinematographic operations. Second, leveraging model distillation based on the Phased Consistency Model, it significantly accelerates inference speed and enhances interaction responsiveness, overcoming computational inefficiency challenges in dynamic scene generation. Additionally, the model achieves high-fidelity dynamics by balancing detailed scene fidelity with long-sequence consistency, surpassing traditional methods that struggle with flickering or abrupt transitions. These advancements position Hunyuan-GameCraft as a robust solution for real-time, interactive game environments requiring both precision and realism.
